# Contributing

Contributions are welcome in data research, corrections, documentation, design,
accessibility, security, translations, and software.

## Choose the right repository

- Listing or correction: `PerkCommons/data`
- Website behavior: `PerkCommons/site`
- Policy or documentation: `PerkCommons/docs`
- Brand asset or token: `PerkCommons/branding`

Open a focused issue before a large change. Use Conventional Commits, keep pull
requests narrow, explain the user or contributor impact, and include tests or
validation appropriate to the risk. By participating, follow the Code of Conduct.

Never include credentials, personal data, confidential correspondence, referral
links, or unlawfully obtained evidence. Disclose provider affiliations and
financial conflicts.
